## 📦 the repos

### 1. [obra/superpowers](https://github.com/obra/superpowers) — ⭐ 160K+

adds superpowers to Claude Code: deep code analysis, auto-refactor, project-wide editing.  
works as a layer on top of the official CLI — zero setup

**replaces:** paid dev productivity tools

---

### 2. [microsoft/markitdown](https://github.com/microsoft/markitdown) — ⭐ 38K+

convert **any file** to clean Markdown for Claude context.  
supports PDF, Word, Excel, PowerPoint, images, HTML, audio

**replaces:** PDF/doc parsing APIs — `$50/mo`

---

### 3. [anthropics/claude-code-skills](https://github.com/anthropics/claude-code-skills) — official

official Anthropic Skills framework (SKILL.md pattern).  
teach Claude to handle documents, automations, and workflows without errors

**replaces:** automation platforms — `$50/mo`

---

### 4. [anthropics/model-context-protocol](https://github.com/anthropics/model-context-protocol) — ⭐ 11K+

official MCP standard — connect Claude to any API, database, Slack, GitHub, Notion.  
hundreds of ready-made servers in the ecosystem

**replaces:** Zapier / Make — `$200/mo`

---

### 5. [Alishahryar1/free-claude-code](https://github.com/Alishahryar1/free-claude-code) — ⭐ 2K+

run Claude Code **completely free** via GitHub Models API.  
step-by-step guide + setup scripts. trending #1 on GitHub April 2025

**replaces:** Claude Pro subscription — `$20/mo`

---

### 6. [HKUDS/RAG-Anything](https://github.com/HKUDS/RAG-Anything) — ⭐ 12K+

multimodal RAG for Claude: texts, tables, images, graphs.  
accurate answers over your own documents in 6 lines of code

**replaces:** paid RAG platforms — `$100/mo`

---

### 7. [simonw/files-to-prompt](https://github.com/simonw/files-to-prompt) — ⭐ 3K+

pack your entire codebase into one optimized Claude prompt.  
supports `.gitignore`, recursive folders, file type filters

**replaces:** context management tools — `$20/mo`

---

### 8. [punkpeye/awesome-mcp-servers](https://github.com/punkpeye/awesome-mcp-servers) — ⭐ 27K+

500+ ready-made MCP servers: browsers, databases, GitHub, Slack, email, finance, maps.  
the most complete MCP ecosystem catalog

**replaces:** scattered SaaS integrations — `$150/mo`
