# 🧠 claude-repos — 8 GitHub repos that save 100+ hours working with Claude

> a curated list of the best open-source tools for building with Claude AI

---

## why this exists

half your time with Claude goes to setup, file prep, finding integrations.  
these 8 repos fix exactly that.

---

## the list

### 1. [obra/superpowers](https://github.com/obra/superpowers) ⭐ 160K+
**replaces:** manual Claude Code environment setup  
adds deep code analysis, auto-refactor, and project-wide editing as a layer on top of official Claude Code

---

### 2. [microsoft/markitdown](https://github.com/microsoft/markitdown) ⭐ 38K+
**replaces:** paid document parsers  
convert any PDF, Word, Excel, PowerPoint, image → clean Markdown for Claude context. zero config

---

### 3. [anthropics/claude-code-skills](https://github.com/anthropics/claude-code-skills) ⭐ official
**replaces:** trial and error with Claude automation  
official Anthropic SKILL.md framework — patterns that teach Claude to handle docs, integrations, and workflows without errors

---

### 4. [anthropics/model-context-protocol](https://github.com/anthropics/model-context-protocol) ⭐ 11K+
**replaces:** $200+/mo in paid integrations  
official MCP standard — connect Claude to any API, database, Slack, GitHub, Notion. plug and play

---

### 5. [Alishahryar1/free-claude-code](https://github.com/Alishahryar1/free-claude-code) ⭐ 2K+
**replaces:** $20/mo Claude subscription  
run Claude Code completely free via GitHub Models API. step-by-step guide + setup scripts. trending #1 on GitHub

---

### 6. [HKUDS/RAG-Anything](https://github.com/HKUDS/RAG-Anything) ⭐ 12K+
**replaces:** $100/mo paid RAG tools  
multimodal RAG framework for Claude — texts, tables, images, graphs. accurate answers over your own documents in 6 lines of code

---

### 7. [simonw/files-to-prompt](https://github.com/simonw/files-to-prompt) ⭐ 3K+
**replaces:** manual context preparation  
pack your entire codebase into one optimized Claude prompt. supports .gitignore, recursive folders, file filters

---

### 8. [punkpeye/awesome-mcp-servers](https://github.com/punkpeye/awesome-mcp-servers) ⭐ 27K+
**replaces:** hours of searching for integrations  
500+ ready-made MCP servers — browsers, databases, GitHub, Slack, email, finance, maps. the complete ecosystem catalog
